Алгоритм поиска максимального числа в массиве:
1. Инициализировать переменную maxVal с первым элементом массива arr[0].
2. Для каждого индекса i от 1 до (N - 1), где N - размер массива:
     1. Если arr[i] больше чем значение maxVal, обновить maxVal значением arr[i].
3. Вернуть maxVal как результат.

Оценка:
- Шаг 1 требует постоянное время O(1).
- Шаг 2 выполняется для каждого индекса от 1 до (N - 1), т.е. (N - 1) раз.
- В худшем случае, каждый элемент arr[i] сравнивается с текущим максимальным значением, что занимает O(1) операций.
- Итоговая оценка времени выполнения алгоритма составляет O(N), где N - размер массива.

function findMax(arr) {
    let maxVal = arr[0]; // O(1)
    for (let i = 1; i < arr.length; i++) { // N - 1 раз
        if (arr[i] > maxVal) {  // O(1)
            maxVal = arr[i];
        }
    }
    return maxVal;
}
сложность алгоритма O(N)
